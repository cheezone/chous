# Test Samples

This directory contains real project examples used to test different preset configurations. These projects are reference projects that are expected to **pass** all checks using their corresponding presets.

## Test Projects

### nuxt.com
- **Purpose**: Test the `@presets/nuxt4.chous` preset.
- **Source**: `third/nuxt.com`
- **Project Type**: Nuxt 4 project.

### project-layout
- **Purpose**: Test the `go.chous` preset.
- **Source**: `third/project-layout`
- **Project Type**: Standard Go project layout.

### ai-chatbot
- **Purpose**: Test the `@presets/nextjs.chous` preset.
- **Source**: https://github.com/vercel/ai-chatbot.git
- **Project Type**: Next.js project.

### pythondotorg
- **Purpose**: Test the `python.chous` preset.
- **Source**: https://github.com/python/pythondotorg.git
- **Project Type**: Python/Django project.

## Usage

These sample projects ensure that the preset configuration files accurately validate the structure of real-world projects. When running tests, ensure these projects pass the lint checks for their respective presets. If a sample project fails to pass, the preset configuration may need adjustment.
